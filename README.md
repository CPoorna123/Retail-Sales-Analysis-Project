# Retail Sales Performance Analytics Dashboard

## Project Overview

This project analyzes retail sales data to uncover insights about **sales performance, profitability, customer segments, and product trends**.

The data was first **cleaned and prepared using Python and Excel**, then visualized using **Power BI** to build an interactive business dashboard.

This project demonstrates **end-to-end data analysis workflow**, starting from data cleaning to building a professional business dashboard.

* **Total Sales:** 1.06M
* **Total Profit:** 120K
* **Total Orders:** 2269
* **Total Quantity Sold:** 17K


## Key Insights

* **Consumer segment** contributes the highest sales.
* **West region** generates the highest revenue.
* Some sub-categories show **negative profit**, indicating loss-making products.
* A few products contribute significantly to total revenue.


## Dashboard Features

* Monthly sales trend analysis
* Sales by region analysis
* Customer segment analysis
* Product-level sales performance
* Profit analysis by sub-category


## Tools & Technologies

* **Python** – Data cleaning and preprocessing
* **Excel** – Initial data exploration and transformation
* **Power BI** – Interactive dashboard and data visualization


## Data Analysis Workflow

1. Collected retail sales dataset.
2. Cleaned and processed data using **Python (Pandas)**.
3. Performed additional formatting and validation in **Excel**.
4. Built an interactive **Power BI dashboard** for visualization.
5. Generated insights for business decision-making.
